# Loja com Arduino

Projeto de uma loja utilizando Arduino Uno.

## Funcionalidades

- Menu animado
- LCD I2C
- Navegação com botões
- Carrinho de compras
- Sistema de pagamento
- Buzzer
- LEDs de confirmação

## Componentes

- Arduino Uno
- LCD I2C
- Botões
- Buzzer
- LEDs
